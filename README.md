# Kanban backend
API for a kanban board system.

## Features

- ### Auth
  - [x] JWT based authentication
  - [ ] Token unvalidation on user logout/delete
  - [ ] External authentication providers (OAuth2)
- ### Boards
  - [ ] Privacy settings
- ### Lists
  - [ ] Automation
- ### Cards
  - [x] Labels
  - [ ] Task list
- ### Misc.
  - [ ] Rate limiting
  - [ ] CORS
  - [ ] Documentation
  - [ ] Docker secrets support (reading config vars from files)
  - [ ] Integration tests
  - [ ] Audit logging
